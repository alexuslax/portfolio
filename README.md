# Stephanie Destura Portfolio

A modern, responsive personal portfolio built with Next.js (App Router), Tailwind CSS, Framer Motion, and Lucide React icons.

## Features

- Clean minimal UI with glassmorphism styling
- Dark mode enabled by default
- Light/dark toggle
- Smooth scrolling navigation
- Responsive sections for mobile, tablet, and desktop
- Subtle fade and hover animations

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser at:

```text
http://localhost:3000
```

## Production Build

```bash
npm run build
npm run start
```
